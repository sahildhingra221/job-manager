## Run Locally

Clone the project

```bash
  git clone https://github.com/sahildhingra221/job-manager
```

Go to the project directory

```bash
  cd job-manager
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
